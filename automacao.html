Automação Residencial com Arduino

1. Componentes necessários

Hardware

  • Arduino UNO, Nano ou ESP32 (recomendado ESP32/ESP8266 — Wi-Fi embutido)
  • Relés 5V (1, 2, 4 ou 8 canais)
  • Sensor de temperatura/humidade DHT11 ou DHT22
  • Lâmpada + tomada/fio para testes
  • Fonte 5V
  • Jumpers
  • Protoboard

Software

  • Arduino IDE
  • Blynk (mais fácil)
  • MIT App Inventor
  • ESP RainMaker (se usar ESP32)

2. Estrutura do Projeto

O sistema terá:

  • ✔ Controle de iluminação
  • ✔ Monitoramento de temperatura
  • ✔ Controle remoto de dispositivos
  • ✔ Comunicação Wi-Fi

3. Esquema Elétrico (básico)

DHT22

  • VCC → 3.3V
  • GND → GND
  • DATA → GPIO 4

Relé

  • VCC → 5V
  • GND → GND
  • IN1 → GPIO 5
  • IN2 → GPIO 18

⚠ Atenção: A lâmpada AC deve ser usada com segurança. Para testes, use LED 12V.

4. Código Completo (ESP32 + Blynk)

#define BLYNK_PRINT Serial
#include <WiFi.h>
#include <WiFiClient.h>
#include <BlynkSimpleEsp32.h>
#include "DHT.h"

#define DHTPIN 4
#define DHTTYPE DHT22
#define rele1 5
#define rele2 18

char auth[] = "SEU_TOKEN_BLYNK";
char ssid[] = "SUA_WIFI";
char pass[] = "SENHA_WIFI";

DHT dht(DHTPIN, DHTTYPE);

void setup() {
  Serial.begin(115200);
  Blynk.begin(auth, ssid, pass);

  pinMode(rele1, OUTPUT);
  pinMode(rele2, OUTPUT);

  digitalWrite(rele1, HIGH);  
  digitalWrite(rele2, HIGH);

  dht.begin();
}

void loop() {
  Blynk.run();

  float t = dht.readTemperature();
  float h = dht.readHumidity();

  Blynk.virtualWrite(V1, t);
  Blynk.virtualWrite(V2, h);

  delay(2000);
}

BLYNK_WRITE(V3) {
  int estado = param.asInt();
  digitalWrite(rele1, !estado);
}

BLYNK_WRITE(V4) {
  int estado = param.asInt();
  digitalWrite(rele2, !estado);
}

5. Criando o APP no Blynk

  • Crie projeto no app Blynk
  • Adicione indicadores (Gauge) para temperatura V1 e umidade V2
  • Adicione botões para relés V3 e V4
  • Copie o Auth Token para o código
